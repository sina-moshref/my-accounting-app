# سامانه مدیریت حسابداری و طرف حساب

## معرفی

سامانه مدیریت حسابداری و طرف حساب یک اپلیکیشن تحت وب برای ثبت، مدیریت و نگهداری اطلاعات اشخاص، مشتریان و طرف‌های حساب است. این پروژه با تمرکز بر تجربه کاربری مدرن، رابط کاربری واکنش‌گرا و معماری قابل توسعه طراحی شده است.

---

## قابلیت‌ها

### مدیریت طرف حساب

- ثبت اطلاعات اشخاص و طرف‌های حساب
- ویرایش اطلاعات ثبت‌شده
- حذف اطلاعات با پنجره تایید (Confirmation Modal)
- نمایش اطلاعات در قالب جدول داده (Data Table)

### رابط کاربری مدرن

- طراحی Responsive برای موبایل و دسکتاپ
- پشتیبانی از حالت روشن (Light Mode) و تاریک (Dark Mode)
- منوهای شناور و سایدبار تعاملی
- استفاده از انیمیشن‌ها و افکت‌های نرم برای بهبود تجربه کاربری

### مدیریت وضعیت برنامه

- استفاده از Redux برای مدیریت State سراسری
- مدیریت عملیات CRUD به صورت متمرکز

### مسیریابی

- پیاده‌سازی مسیریابی سمت کاربر (Client Side Routing)
- صفحات اختصاصی:
  - صفحه اصلی
  - مدیریت طرف حساب
  - صفحه ورود
  - صفحه 404
